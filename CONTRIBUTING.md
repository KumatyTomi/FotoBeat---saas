# Contributing to FotoBeat SaaS

Thanks for your interest in contributing to **FotoBeat SaaS**! This project aims to build a web‑based service for automatically generating music videos from photos and audio. We welcome contributions of all kinds — not only code.

## How to contribute

1. **Fork** the repository and create a new branch (`git checkout -b feature/my-feature`).
2. Run the project locally to test your changes:

   ```bash
   npm install
   npm run dev
   ```

   The project uses Vite and React. See the README for production build commands.

3. **Commit** your changes with clear messages and push to your fork.
4. **Open a pull request** against the `main` branch.

## Guidelines

- Follow the [Contributor Covenant Code of Conduct](./CODE_OF_CONDUCT.md).
- Keep documentation up to date. If you add or change features, update the README or create additional docs.
- Be respectful when discussing issues or reviewing pull requests.
- Besides coding, you can contribute by designing templates, improving translations, triaging issues and helping with user support.

Thank you for helping make FotoBeat SaaS better!
